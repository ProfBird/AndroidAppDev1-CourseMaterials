Android Application Development 1

by Brian Bird


Course Description

In this course, you will be introduced to mobile application programming concepts and mobile application development methodology using the Android Software Development Kit, Java, and Android Studio.

Learning Objectives

Design the core logic for a mobile software application. Write, debug, and test the code for the core logic for an application. Design User Interfaces that adapt to multiple screen sizes. Evaluate mobile app designs and architectures in terms of user experience, performance, and maintainability.

Workload

  • 40 hours of class lecture/lab time
  • 80 hours reading and doing lab assignments outside of class

This follows the general guideline for college courses-- that you should spend two hours working outside of class for every hour of class time.

Software

You are expected to use your own computer to complete the assignments for this class. You will need to install the following software:

  • Java SE 8: If you don't already have the JDK installed, it will be installed by Android Studio, or you can download it here: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
  • Android Studio: You can download Android Studio, which includes the Android SDK (for Windows or OS-X) here: http://developer.android.com/sdk/. The Android Studio setup program will install everything you need to develop Android applications: a version of the intelliJ IDE customized for Android development, the Android SDK and Platform tools, an Android emulator and system images.

Grade

  • Term project: 50%
  • Lab assignments: 50%

There are 7 Lab assignments which constitute 50% of your grade. The last lab assignment, lab 8, is optional extra credit. There is a term programming project which comprises the other 50% of your grade. Your grade is based on weighted percentages, not total points.

Grades are determined using a straight percentage (not a curve of any kind): 90% is an A, 80% a B, 70% C, 67% C-/P, 60% D, 0-59% F/N. A minimum of 97% is required for the grade of A+.

Term Project

Near the middle of the term you will write a proposal for an Android application that will be your term project. The project will be due the day scheduled for the final exam (there will not be a final exam). The application must have the following:
"    At least 5 kinds of widgets
"    An action bar with at least 2 items on the menu and an up button on the second activity.
"    At least 2 activities (the main activity should launch a second activity)
"    Different layouts for landscape and portrait orientation (using fragments is optional)
"    Persist activity state when rotating
Your proposal should include at least a two paragraph description of the app, a list of features. and simple UI diagrams for the two activities.
